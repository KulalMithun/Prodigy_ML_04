    <h2>Project Aim</h2>
    <p>Hand gesture recognition using machine learning techniques.</p>
    
    <h2>Dataset</h2>
    <p>The project uses the <a href="https://www.kaggle.com/datasets/gti-upm/leapgestrecog" target="_blank">Leap Gesture Recognition Dataset</a> from Kaggle.</p>
    
    <h2>Implementation</h2>
    <ul>
        <li>Built using Keras library</li>
        <li>Achieved <strong>99% validation accuracy</strong></li>
        <li>Added live prediction using OpenCV integrated with the ML model</li>
    </ul>
    
    <h2>How to Run Live Prediction</h2>
    <p>The project includes a live gesture recognition feature using the webcam:</p>
    <ol>
        <li>Run the prediction script</li>
        <li>The system will start recognizing hand gestures in real-time</li>
        <li>To stop the live prediction, press <code>q</code> on your keyboard</li>
    </ol>
